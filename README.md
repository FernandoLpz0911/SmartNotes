SmartNotes Project

What it does:
Takes user notes and
- Summarizes it
- Applies relevant tags
- Allows semantic searching (using AI)
- Potentailly allowing voice to text input and transcript summarizing

Why it matters:
- Speeds up the process of note-taking
- Organizes note-taking
- Makes searching through notes significantly faster

Tech Stack:
Frontend = React
Backend = Python, FastAPI, SQLite
AI/ML Components = Text Summarizing, Keyword extraction and tagging, Search understanding (allows it to be semantic)

Things To Take Away:
- Python Language
- API Integration
- ML Libraries
- Data Handling
- AI concepts

Expansion Features:
- AI organizing based on topics
- Voice Support
- Chatbot Q&A for studying over notes
